# AI 翻译助手

一个轻量级的 AI 翻译桌面小工具，基于 Electron 开发。

## 功能特性

- 🔄 **剪贴板监控** - 自动检测复制的文字并翻译
- ⌨️ **快捷键翻译** - `Ctrl+Shift+T` 翻译剪贴板，`Ctrl+Shift+R` 翻译并替换选中文字
- 💬 **AI 对话** - 支持多轮对话，可保存历史记录
- 🖼️ **图片识别** - 支持发送图片进行识别
- 🌍 **多语言** - 支持中/英/日/韩/法/德等语言互译
- 📌 **窗口置顶** - 可固定在屏幕最前方
- 🎨 **简洁界面** - 无边框透明窗口，支持拖拽调整大小

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行

```bash
npm start
```

### 打包

```bash
npm run build
```

## 配置

首次使用需要配置 API：

1. 点击右上角设置按钮
2. 填写 API 地址（兼容 OpenAI 格式）
3. 填写 API Key
4. 选择模型名称
5. 点击测试连接，成功后保存

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+T` | 翻译剪贴板内容 |
| `Ctrl+Shift+R` | 翻译选中文字并替换 |
| `Ctrl+Enter` | 翻译输入框内容 |

## 技术栈

- Electron 28
- electron-store（配置持久化）
